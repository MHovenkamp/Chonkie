Detailed Description

freqMeasuringtool ADT

This is an ADT that gives the user a frequenty from an analog to digital converter pin. The frequenty in microseconds is stored as an integer inside the function the ADT is dependant on the hwlib library for an input pin_in and its now_us() function. https://github.com/wovo/hwlib
